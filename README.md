# 🚦 Road Accident Analysis Dashboard

This project is an interactive Excel dashboard created to analyze road accident data and extract meaningful insights.

---

## 📊 Key Features

- Total casualties overview
- Severity analysis (Fatal, Serious, Slight)
- Monthly trend comparison (CY vs PY)
- Casualties by road type and surface
- Area-wise (Urban/Rural) analysis
- Light condition analysis (Day/Night)
- Interactive filters (Date, Weather, Wind)

---

## 📈 Insights

- Most accidents occur on single carriageways
- Urban areas have higher casualties compared to rural
- Daylight accidents are more frequent than night
- Slight severity cases are significantly higher than fatal

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Slicers (Filters)
